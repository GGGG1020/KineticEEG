Welcome to KineticEEG

This package is a open source brain computer interface. Using an innovative and intelligent algorithms, we are able to make reliable classifications.

Compatibility

KineticEEG is currently compatible with the Emotiv EPOC headset.You will need Windows, an Emotiv Headset, and a Research or Education License from Emotiv(Not Developer's Edition).

Using KineticEEG

KineticEEG can be used as a library in your brain computer interface projects. It also comes with a plethora of tools for doing research, saving data, and working on projects.
